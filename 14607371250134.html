AutoLayout

2016/4/16

苹果在 iOS6 中首次引入 AutoLayout 技术。

1. 屏幕尺寸

用于描述屏幕尺寸的相关参数有很多，很容易混淆。我们首先将这些参数分为两类，即物理参数和逻辑参数。

物理参数有：
* 屏幕尺寸（Screen sizes），单位一般是英寸（inches），取得是屏幕对角线的长度。
* 像素（pixels），一个不可再分割的单色点（R/G/B）。
* 分辨率（resolutions）

2. 约束（Constraint）

2.1 约束方程

view1.attr1 = multiplier * view2.attr2 + constant

2.2 创建约束

iOS6 之后 UIKit 中引入 NSLayoutConstraint 类来表示约束。当我们使用 AutoLayout 时，首先要先创建约束对象，目前有两种构建 NSLayoutConstraint 对象的方式：

方式一：直接传入约束方程需要的各项参数

+ (instancetype)constraintWithItem:(id)view1
                         attribute:(NSLayoutAttribute)attr1
                         relatedBy:(NSLayoutRelation)relation
                            toItem:(id)view2
                         attribute:(NSLayoutAttribute)attr2
                        multiplier:(CGFloat)multiplier
                          constant:(CGFloat)c
NSLayoutAttribute 说明
NSLayoutAttributeLeft 左
NSLayoutAttributeRight 右
NSLayoutAttributeTop 顶
NSLayoutAttributeBottom 底
NSLayoutAttributeLeading
NSLayoutAttributeTrailing
NSLayoutAttributeWidth 宽
NSLayoutAttributeHeight 高
NSLayoutAttributeCenterX 水平方向中心
NSLayoutAttributeCenterY 竖直方向中心
NSLayoutAttributeBaseline 字体基线
NSLayoutAttributeLastBaseline
NSLayoutAttributeFirstBaseline iOS8 新增
NSLayoutAttributeLeftMargin iOS8 新增
NSLayoutAttributeRightMargin iOS8 新增
NSLayoutAttributeTopMargin iOS8 新增
NSLayoutAttributeBottomMargin iOS8 新增
NSLayoutAttributeLeadingMargin iOS8 新增
NSLayoutAttributeTrailingMargin iOS8 新增
NSLayoutAttributeCenterXWithinMargins iOS8 新增
NSLayoutAttributeCenterYWithinMargins iOS8 新增
NSLayoutAttributeNotAnAttribute 值为0
NSLayoutRelation 说明
NSLayoutRelationLessThanOrEqual 小于等于
NSLayoutRelationEqual 等于
NSLayoutRelationGreaterThanOrEqual 大于等于

方式二：VFL

VFL 是 Visual Format Language 的缩写。VFL 语法参考这里。

+ (NSArray<__kindof NSLayoutConstraint *> *)constraintsWithVisualFormat:(NSString *)format 
                                                                options:(NSLayoutFormatOptions)opts 
                                                                metrics:(nullable NSDictionary<NSString *,id> *)metrics 
                                                                  views:(NSDictionary<NSString *, id> *)views;

2.3 添加约束

使用 UIView 的 addConstraint 方法为视图添加一项约束：

- (void)addConstraint:(NSLayoutConstraint *)constraint NS_AVAILABLE_IOS(6_0);

注意添加了约束的视图需要将 translatesAutoresizingMaskIntoConstraints 属性设置为 NO。

2.4 注意事项

  1. 必须是父视图调 addConstraint 添加与子视图的约束，不能反过来。否则会导致 The view hierarchy is not prepared for the constraint Crash
